# Business Glossary

## Overview
This document provides a business glossary for the Retail Data Analytics project, defining key terms, metrics, and KPIs used across the data warehouse. This ensures consistent understanding between technical and business stakeholders.

---

## General Terms

| Term | Definition | Example |
|------|------------|---------|
| **Medallion Architecture** | A data design pattern with three layers (Bronze, Silver, Gold) that progressively refine data | Bronze → Silver → Gold |
| **Data Warehouse** | A centralized repository for structured data optimized for reporting and analytics | Snowflake |
| **ETL** | Extract, Transform, Load - process of moving data from sources to destination | Python script |
| **ELT** | Extract, Load, Transform - data loaded first, then transformed | Snowflake Views |
| **Star Schema** | A dimensional model with a central fact table surrounded by dimension tables | Gold layer design |
| **Fact Table** | A table containing quantitative business metrics | fact_sales |
| **Dimension Table** | A table containing descriptive attributes | dim_customers |
| **Surrogate Key** | A unique identifier generated for join purposes | customer_key |
| **Natural Key** | A key that has business meaning | customer_id |

## Sales Metrics

| Metric | Definition | Formula | Unit |
|--------|------------|---------|------|
| **Gross Revenue** | Total revenue before any deductions | SUM(revenue) | Currency |
| **Net Revenue** | Revenue after deducting discounts | revenue - discounts | Currency |
| **Total Discounts** | Sum of all discount amounts | SUM(discounts) | Currency |
| **Units Sold** | Quantity of products sold | SUM(units_sold) | Integer |
| **Net Units Sold** | Units sold minus returns | units_sold - COALESCE(returns, 0) | Integer |
| **Average Selling Price** | Average price per unit sold | revenue / NULLIF(units_sold, 0) | Currency |
| **Realized Price** | Net revenue per net unit sold | net_revenue / NULLIF(net_units_sold, 0) | Currency |

## Business Rules

1. **Net Revenue Calculation**
   - Net Revenue = Gross Revenue - Discounts
   - If discounts are NULL, treat as 0

2. **Net Units Calculation**
   - Net Units Sold = Units Sold - Returns
   - If returns are NULL, treat as 0

3. **Margin Calculation**
   - Unit Margin = Unit Price - Unit Cost
   - Margin % = (Unit Price - Unit Cost) / Unit Cost
   - If unit cost is 0, margin is NULL (undefined)

4. **Sell-Through Rate**
   - Units Sold / Ending Inventory
   - If ending inventory is 0, rate is NULL

5. **Forecast Accuracy**
   - (Actual - Forecasted) / Forecasted * 100
   - If forecasted is 0, accuracy is NULL
